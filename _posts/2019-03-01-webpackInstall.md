全局安装 webpack 命令：

npm install -g webpack

安装特定版本 webpack（比如 3.x.x）：

npm install -g [webpack@3](mailto:webpack@3)

看 webpack 是否安装成功（下面命令都可以）：

webpack -v

webpack --version
